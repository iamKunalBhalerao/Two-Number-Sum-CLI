## Setup Locally

Clone the project

```bash
  git clone https://github.com/iamKunalBhalerao/Two-Number-Sum-CLI.git
```

Go to the project directory

```bash
  cd Two-Number-Sum-CLI
```

Install all the dependancy

```bash
  npm install
```

Give execute permission

```bash
  chmod +x app.js
```

Install Locally

```bash
  npm link
```


## Usages

#### Run It in terminal

```http
  twosum
```
